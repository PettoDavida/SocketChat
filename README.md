# SocketChat

## About

SocketChat is a webchat built with socket.io. SocketChat consists of rooms where users can chat. If a room is empty it will be deleted.

## Build Project

open terminal inside the project root folder and run

```
npm i
```

## Start project

open terminal inside the project root folder and run

```
npm run dev
```
